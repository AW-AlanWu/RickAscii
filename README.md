# RickAscii

目前做法是把影片用KMplayer之類的播放軟體逐幀截圖，然後再用Ascgen批量把圖片轉成Ascii Art的txt檔，最後用C++讀檔顯示出來。<br>
因為是用console輸出的方式，所以看起來會有點卡頓。<br>
然後因為有人建議不要用Windows的cmd，自己寫gui才比較不會卡頓，所以目前正在設計<br>(明明只是個meme挑戰，我是在認真什麼XD)<br>
